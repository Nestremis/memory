# memory game

### Repozytorium zawiera pliki z katalogu SRC. 
### W gotowym środowisku CreateReactApp podmieniamy pliki z katalogu SRC na te z repozytorium.

### W input wpisujemy pary słów rodzielonych znakiem równości np. word=słowo i wciskamy enter. 
### Generuje to 2 karty rzucone w losowej kolejności na ekranie. Stanowią one parę. Kliknięcie karty powoduje jej odwrócenie na 2s. 
### Jeśli w tym samym czasie odsłonięta jest karta z pary, pozostaną one odwrócone do końca gry.
